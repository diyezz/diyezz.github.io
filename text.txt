Гра на JavaScript функцій: Пошук скарбів

Ласкаво просимо до "Пошуку скарбів"! У цій грі ви будете використовувати різні типи функцій JavaScript для навігації через загадковий острів та виявлення схованих скарбів. Ваша мета - використовувати функції об'явлення, виразів функцій та стрілкові функції, щоб завершити завдання та знайти остаточний скарб.

Інструкції:

Завдання на Function Declaration :
Напишіть об'явлення функції під назвою moveForward, яка виводить "Переміщено вперед" у консоль.

Завдання на Function Expression:
Напишіть вираз функції під назвою turnLeft, який виводить "Повернуто ліворуч" у консоль.

Завдання на Arrow Function:
Напишіть стрілкову функцію під назвою turnRight, яка виводить "Повернуто праворуч" у консоль.

Завдання на навігацію:
Використовуйте створені функції для навігації по острову. Ви отримаєте серію інструкцій:

"F": Переміщення вперед
"L": Поворот ліворуч
"R": Поворот праворуч
Навігуйте по острову, використовуючи ці інструкції, щоб відкрити приховані сліди та скарби.
Бонусне завдання:
Реалізуйте обробку помилок для обробки недійсних інструкцій та уникнення загублення гравця на острові.

Приклад:

javascript
Copy code
moveForward(); // "Переміщено вперед"
turnLeft();    // "Повернуто ліворуч"
turnRight();   // "Повернуто праворуч"
Удачі, і нехай вам пощастить відкрити скарби острова!
